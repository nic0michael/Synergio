Form | View Records

How to Use

1. Go to the Form page to enter customer and vehicle details.

2. Click “Save Record” to store the entry in service_records.csv.

3. Go to the View Records to view captured service records.

4. The data can be opened in Excell as a spreadsheet after reformatting the columns

File Storage

All service data is stored locally in a file called service_records.csv.
